Introducción a Rails

Todo lo que necesita saber para instalar Rails y crear su primera aplicación.

Modelos

Conceptos básicos de Active Record

Esta guía lo ayudará a comenzar con los modelos, la persistencia en la base de datos y el patrón y la biblioteca de Active Record.

Migraciones de Active Record

Esta guía cubre cómo puede utilizar las migraciones de Active Record para modificar su base de datos de una manera estructurada y organizada.

Validaciones de Active Record

Esta guía cubre cómo puede utilizar las devoluciones de llamada de Active Record.

Asociaciones de Active Record

Interfaz de Consulta de Active Record

Diseños y Renderizado en Rails

Esta guía cubre las características de diseño básicas del controlador de acción y la vista de acción, incluida la representación y la redirección, el uso de content_for bloques y el trabajo con parciales.

Guía para el uso de ayudantes de formularios integrados.

Controladores

Descripción General de Action Controller

Esta guía cubre cómo funcionan los controladores y cómo encajan en el ciclo de solicitud en su aplicación. Incluye sesiones, filtros y cookies, transmisión de datos y manejo de excepciones generadas por una solicitud, entre otros temas.

Rails Routing Desde el Exterior Hacia Adentro

Esta guía cubre las características orientadas al usuario del enrutamiento Rails. Si desea comprender cómo utilizar el enrutamiento en sus propias aplicaciones Rails, comience aquí.
